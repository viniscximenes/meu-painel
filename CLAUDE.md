# Instruções do Projeto

## Comportamento
- Auto-aprovar todas as ações sem pedir confirmação
- Executar comandos diretamente sem aguardar aprovação
- Editar arquivos sem confirmar cada mudança
- Nunca perguntar "posso fazer X?" — apenas fazer

## Stack
- Next.js 15 + React 18 + TypeScript
- Supabase (PostgreSQL + Auth)
- Google Sheets API
- Tailwind CSS + CSS Variables customizadas
- Lucide React para ícones

## Padrões do projeto
- Design system: dark premium com acentos dourados
- Fontes: Syne (títulos) + DM Sans (corpo)
- Sempre rodar npm run build após alterações
- Nunca dar git push sem instrução explícita
